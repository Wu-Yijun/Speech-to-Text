# Voice Input

A simple speech-to-text web using Chrome's Speech Recognition API

[**Web Page:** Wu-Yijun.github.io/Speech-to-Text](https://Wu-Yijun.github.io/Speech-to-Text)

- Fast, Real time, Accurate.
- Multiple Language support, Editable.
- Keyeboard Shortcuts, Easy to use.
- Poor support for punctuations.
